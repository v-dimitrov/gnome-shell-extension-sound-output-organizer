## Online install
- Online install: this extension is not yet available on [GNOME Extensions Website](https://extensions.gnome.org/).

## Local install
1. Create new folder:
   - $HOME/.local/share/gnome-shell/extensions/sound-output-organizer@sound-output-organizer
2. Put the contents of this repository inside it
3. Logout and Login again
4. Open up your extensions app and enable `Sound Output Organizer` extension
5. Go to settings of the extension to rename/hide any of your audio devices

## Uninstall
To bring back original sound device names you can do any of these:
- Delete the custom names from extension settings (automatically brings back the original names)
- Disable the extension (removes all customizations)
- Uninstall the extension (removes all customizations)
